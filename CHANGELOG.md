# Changelog

All notable manual changes to this API collection are documented here.

## 2026-06-26

### Added
- **Warnely** to **Open Data** — Composite travel-safety scores for 180 countries combining FCDO, US State, GPI, WGI, and live incidents (CC BY 4.0, OpenAPI 3.1). Auth: none, HTTPS: yes, CORS: yes. ([#5](https://github.com/gunjanjaswal/awesome-public-free-apis/pull/5))
- **BGPT** to **Science & Math** — Scientific paper search with structured full-text evidence (methods, limitations, COI, quality scores). Auth: none, HTTPS: yes, CORS: yes. ([#6](https://github.com/gunjanjaswal/awesome-public-free-apis/issues/6))
